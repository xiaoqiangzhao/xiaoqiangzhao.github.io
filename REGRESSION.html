Regression¶

Overview¶

This tool is designed for local regression mission, which can be controlled and checked much more easy. See Regression Code.

QuickStart¶

  1. Invoke you python shell or ipython

  2. call setup script

    1. for rtl regression(setup.py)

      >>> run setup.py <your_test_plan.xlsx>

    2. for power aware regression(setup_pa.py)

      >>> run setup_pa.py <your_test_plan.xlsx>

  3. you then will get object re for further operations.

RelavantModule¶

Dependency¶

Required Modules:
  1. termcolor
  2. openpyxl

LocalModule¶

PlanAnalysis¶

This module is used to analysis Freescale-Style XLSX based test plan. For any other sytle, you can extend from this class and override method self.parser
Usage¶

Example code

import sys
from PlanAnalysis import XlsxParser
import sys
xlsx_file = sys.argv[1]
test = XlsxParser(xlsx_file)
test.parser()

Here, we will generate an json file as the result, we can also refer to that file by test.json_file for further use.

Source Code Ref¶

  1. setup.py(download).
#! /home/b51816/local/bin/python3
from SubmitJobs import SubmitJobs
from PlanAnalysis import XlsxParser
import sys
xlsx_file = sys.argv[1]
test = XlsxParser(xlsx_file)
test.parser()

re = SubmitJobs(test.json_file)
re.generate_compile_sh()
for v in re.vectors:
    if not v == 'null' :
        re.generate_simulate_sh(v)
re.gen_results()
  1. setup_pa.py(download)
#! /home/b51816/local/bin/python3
from SubmitJobs import SubmitJobs
from PlanAnalysis import XlsxParser
import sys
xlsx_file = sys.argv[1]
test = XlsxParser(xlsx_file)
test.parser()

re = SubmitJobs(test.json_file)
re.generate_compile_sh(lps=True)
for v in re.vectors:
    if not v == 'null' :
        re.generate_simulate_sh(v,lps=True)
re.gen_results()
